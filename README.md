# Netflix EDA Project

**Exploratory Data Analysis on Netflix Dataset using Python**

## **Project Overview**
This project analyzes the Netflix dataset (~9000+ titles) to extract insights about **content types, genres, release years, and trends**.  
The goal is to understand what kind of content is most popular, how Netflix releases titles over the years, and visualize key patterns.

## **Tools & Libraries Used**
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## **Dataset**
- Dataset: `netflix_titles.csv`  
- Source: Kaggle 
- Contains columns like `title`, `genre`, `release_year`, `country`, `rating`, etc.

## **Analysis Performed**
- Data cleaning & preprocessing (handling missing values, duplicates)  
- Top genres, content types, and countries analysis  
- Monthly & yearly release trends  
- Visualization using bar charts, line charts, and pie charts

## **Key Insights**
- Most titles are in **Dramas** and **Comedies**  
- **United States** has the highest number of Netflix titles  
- Recent years show an **increase in TV Shows**  
- Visualizations provide easy-to-understand patterns for stakeholders
